# Excel Sales Dashboard

## Tools Used
Microsoft Excel (Pivot Tables, Charts, VLOOKUP)

## Project Description
Created an interactive sales dashboard to analyze performance and trends.

## Key Features
- Pivot table analysis
- Monthly sales tracking
- Product-wise performance
- Interactive charts
